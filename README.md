# Mealplanner

## Project structure

```
P2/
├── client/                 
│   └── src/
│       ├── app/            
│       ├── components/
│       │   ├── layout/     
│       │   └── recipes/    
│       ├── context/        
│       ├── pages/          
│       ├── services/       
│       ├── styles/         
│       └── utils/          
│
└── server/                 
    └── src/
        ├── config/         
        ├── routes/         
        ├── services/       
        └── index.js        
```

## Run locally

**Server** (from `server/`):

```bash
npm start
```

**Client** (from `client/`):

```bash
npm run dev
```

Server: http://localhost:3000  
Client: http://localhost:5173
